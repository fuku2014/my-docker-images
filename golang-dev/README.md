golang-dev
=====================

golang develop env

=======


Usage
-----

### Step 1. Set env ###

```
$ export HTTP_PROXY=http://proxy.example.com:8080 # use proxy
$ export USER_NAME=XXX00000                       # 35id
$ export PASSWORD=XXXXXXXXX                       # pw
```

### Step 2. Build images ###

```
$ docker-compose build
```

### Step 3. Run container ###

```
$ docker-compose up
```
